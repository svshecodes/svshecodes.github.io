Introduction

She.codes is a series of coding workshops
for girls to learn HTML/CSS, AppInventor, and Python.

Learn to create websites, code, and build apps. Through hands-on activities and collaborative projects, you'll learn the fundamentals of computer science while embracing your creativity and having fun!

The Details

She.codes will host 3 exciting workshops in January.


  • 1/7: Website development (Village Square Library)
  • 1/21: App creation (Village Square Library)
  • 1/28: Python (Evergreen Library)

All dates are Sunday, 9 AM - 4 PM.

1/7 Website Development

The first workshop is on website development using HTML and CSS. As they learn about what a programming language really is and are introduced basic tags in HTML, girls can code directly on jsfiddle.net and see their website slowly bloom to life. Once students learn how to use CSS to enhance their website's design and visual appeal, they will be able to work together on a project together: building a website to help solve a social issue.

1/21 App Development

The second workshop is about app making, a useful and creative process, using App Inventor. Girls will learn the inner workings of App Inventor and gain the necessary skills to create their own app. At the end, students will work in a group to build an app from scratch about a social problem in the community.

1/28 Python

In the final workshop, girls will learn programming in Python. Starting with the basics of variables and expressions, and going through more advanced topics like functions and classes, girls will be able to code along in the lesson on repl.it. At the end of the workshop, girls will participate in a coding competition, solving challenging programming problems and testing their skills in Python.
