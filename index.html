Coding Workshops for Girls

Introduction

She.codes is a series of coding workshops
for girls to learn HTML/CSS, AppInventor, and Python.

Learn to create websites, code, and build apps. Through hands-on activities and collaborative projects, discover the beauty of computer science while embracing your creativity and having fun!

The Details

She.codes hosted 3 exciting workshops in January 2018, with coding lessons, a Q&A session with girls working in CS, and free prizes.

Time

The Workshops

1/7 Website Development - Village Square Library

The first workshop teaches website development using HTML and CSS. As participants learn about programming languages and are introduced to basic tags in HTML, they can code directly on jsfiddle.net and see their website slowly bloom to life. Once participants learn how to use CSS to enhance their website's design and visual appeal, they will be able to work together on a project to build a website for social good.

1/21 App Development - Village Square Library

The second workshop teaches app-making using MIT App Inventor. Participants will learn the inner workings of App Inventor and gain the necessary skills to create their own mobile app. At the end, participants will work in a group to build an app from scratch about a social problem in the community.

1/28 Python - Evergreen Library

In the final workshop, girls will learn the how to program in Python. Starting with the basics of variables and data types, and going through more advanced topics like functions and classes, girls will learn important concepts to bolster their programming skills. At the end of the workshop, girls will participate in a coding competition, solving challenging programming problems and testing their abilities in Python.
